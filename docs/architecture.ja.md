## ワールド状態

`SilenceWorld` は，ソート済み `BTreeMap<AgentId, Employee>`（キーのソートが決定論的な活性化順序を保証する）の従業員，チーム表，従業員間 `SocialNetwork`（既定 Watts–Strogatz），争点顕在性 `σ(t)`，全社沈黙風土 `C(t)`，ステップ別報復バッファを保持する．

各 `Employee` は持続的な IVT 認知構造 — `ivt_strength` ι_i ∈ [0,1] と 5 ルール（`target_id`，`need_data`，`no_bypass`，`no_embarrass`，`career_consq`）上の simplex 重み `ivt_rule_weights` — に加え，文脈ドライバ（`fear`，`psych_safety`，`perceived_silence`，`voice_threshold`，`private_concern`），現在の `expression`（VOICE / SILENCE / NEUTRAL），沈黙 `motive`，直近の意思決定で発火した IVT ルール id（`last_active_rules`）を持つ．

## 9 機構 × 6 フェーズ

socsim の固定ループ `PreStep → Environment → Decision → Interaction → Reward → PostStep` に 9 機構を配置する：

| # | 機構 | フェーズ | 役割 |
|---|------|---------|------|
| 1 | `IssueSalience` | Environment | σ(t) を平均回帰；`shock_t` で外生ショックを注入． |
| 2 | `RetaliationEvent` | Environment | 確率 `p_retaliate` で報復被影響者をマーク． |
| 3 | `FearAppraisal` | Decision | 脅威評価：`f_i ← clamp(f_i + α·retaliated − γ·max(u,0))`． |
| 4 | `VoiceDecisionRule` / `VoiceDecisionLlm` | Decision | **排他．** ロジット（rule / rule_no_ivt）vs LLM 5 ルール内省． |
| 5 | `SilenceSpiral` | Interaction | ρ_i ← 近傍沈黙比率（Noelle-Neumann 1974）． |
| 6 | `PrefalseCascade` | Interaction | *偽の* 沈黙者（private_concern ≥ 0）が近傍 VOICE 比率 > θ_i で発言へ転換（Kuran 1995）． |
| 7 | `OrgPerformance` | Reward | チーム知識ストック `K_k(t) ← (1−δ)K_k(t) + voice_share_k`． |
| 8 | `PsafetyUpdate` | PostStep | ψ_i ← clamp(ψ_i + η·voiced − ν·retaliated)． |
| 9 | `ClimateSilence` | PostStep | `C(t)` + チーム別風土を集計；収束デルタを追跡． |

意思決定機構のみが **排他** であり，driver は `--llm-mode` ごとに `VoiceDecisionRule`（`rule` / `rule_no_ivt`）か `VoiceDecisionLlm`（`llm`）のどちらか 1 つだけを挿入する．

## 同期更新

意思決定機構は Decision フェーズ開始時に全従業員の局所文脈（`ρ_i`，`u_{k(i)}`，`σ`，`ψ_i`）をスナップショットし，新しい expression/motive/active-rules をそのスナップショットから一括計算して書き戻す．同一ステップ内で LLM 呼び出しの副作用が他エージェントの観測に漏れない．

## IVT ルール：rule モード vs LLM モード

- **rule / rule_no_ivt** — VOICE ロジット `σ(β0 + β_ψ ψ + β_u u + β_σ σ − β_f f − β_ι ι − β_C ρ + β_θ·cascade)` が VOICE/SILENCE を決める．沈黙者については，5 ルールが各々独立に確率 `σ(k·(ι·w_r − mid))`（ルールごとの一様乱数）で発火するため，ルールは一斉ではなく弁別的に発火する．`rule_no_ivt` は `β_ι = 0` を強制しルールを発火させない（IVT レイヤを ablate）．
- **llm** — 5 ルールを salience `ι_i·w_{i,r}` 付きでプロンプトに提示し，LLM が `{decision, motive, active_rules}` を JSON で返す．

## RNG ストリーム

`derive_seed(root, &[stream])` で乱数を分割する：

- stream `0` — world init（従業員属性 + ネットワーク生成），
- stream `1` — engine（scheduler + 報復ベルヌーイ + rule-vote ベルヌーイ），
- stream `2` — 報復（ドキュメント用ラベル；engine ストリームに統合），
- stream `3` — LLM `(agent_id, t)` seed 根．

## 二層決定論

socsim コアは bit 再現可能；LLM レイヤは `VoiceDecisionLlm` に閉じ込め，`temperature=0` + `(agent_id, t)` seed + `socsim-llm` プロンプト→応答キャッシュで擬似決定論化する．
