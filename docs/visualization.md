# Visualization

[English](visualization.md) | [日本語](visualization.ja.md)

The Python `detert-tools` package reads a results directory and renders PNGs. Install with `uv sync`, then run any subcommand.

## `visualize`

Single-run plots from `metrics.csv` / `rule_activation.csv` / `agents.csv`:

- **silence_timeseries.png** — upward silence rate, silence rate, and climate of silence per step, with the HiCo .50 anchor line.
- **rule_firing_heatmap.png** — the five IVT rules' activation share `a_r` over time as a heatmap.
- **silence_voice_scatter.png** — final-step expression (VOICE / SILENCE / NEUTRAL) plotted over IVT strength ι × private concern b.

```bash
uv run detert-tools visualize --results-dir results/latest
```

## `visualize-sweep`

Sweep plots from `sweep_summary.csv`:

- **sweep_phase_diagram.png** — mean upward silence rate across the β_ι × ψ̄ grid (heatmap).
- **sweep_beta_ivt_curve.png** — upward silence vs β_ι, one line per ψ̄, with the HiCo .50 anchor.

```bash
uv run detert-tools visualize-sweep --results-dir results/<ts>_sweep
```

## `show-experiment-settings`

Pretty-prints `config.json` (run) or `sweep_config.json` (sweep / ablation) plus `llm_meta.json`; `--json` emits machine-readable JSON.

```bash
uv run detert-tools show-experiment-settings --results-dir results/latest
```
