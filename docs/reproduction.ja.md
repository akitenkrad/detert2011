# 再現

[English](reproduction.md) | [日本語](reproduction.ja.md)

`detert-tools reproduce` は Track-B ABM の results ディレクトリ上で end-to-end に走り（調査データ不要），2 つの成果物を生成する．

```bash
cargo run --release -- run --llm-mode rule --t-max 60 --runs 1 --seed 42
uv run detert-tools reproduce --results-dir results/latest
```

| `upward_silence_rate` | ≈ 0.50（Study 1 HiCo 調査） | ±.07 以内 |

silence–voice 相関は実行レベルの **時間平均** per-agent 値（`llm_meta.json` に `silence_voice_corr` として記録）である．`metrics.csv` のステップ別値はステップ内 VOICE/SILENCE スナップショット（厳密な二分 → −1）だが，論文が報告するのは弁別的な silence と voice の傾性間の段階的重複である．

## 2. CFA 系適合度指標（`cfa_fit_indices.csv`）

ツールは `agents.csv`（`active_rules` 列）から N×5 のバイナリ IVT ルール活性化行列を構築し，その相関行列に **単一共通因子** を当てはめ，χ² / df / CFI / RMSEA / SRMR 様の指標を報告する．5 ルールが *弁別的* に発火する（非対角同時発火が低い）ため単一因子の適合は悪く，論文の Study 3 の知見 — ルールは単一の一般沈黙因子に縮退せず多ルール（5 因子）構造を成す（*正しい* モデルで RMSEA = .05, CFI = .97）— を操作化する．

因子負荷量には利用可能なら `factor-analyzer` を用いる．互換性のある NumPy / scikit-learn の下で end-to-end の経路を保つため PCA フォールバックを備える．

最大非対角ルール同時発火（`metrics.csv` 由来）を直接的な弁別統計として併記する．これは 0.50（設計の弁別的妥当性基準）を下回る．
