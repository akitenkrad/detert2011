# Reproduction

[English](reproduction.md) | [日本語](reproduction.ja.md)

`detert-tools reproduce` runs end-to-end on a Track-B ABM results directory (no survey data required) and produces two artefacts.

```bash
cargo run --release -- run --llm-mode rule --t-max 60 --runs 1 --seed 42
uv run detert-tools reproduce --results-dir results/latest
```

## 1. Table-4-style report (`table4_report.csv`)

Steady-state means over the second half of the run, checked against the paper's anchors:

| Indicator | Paper anchor | Reproduction target |
|-----------|--------------|---------------------|
| `upward_silence_rate` | ≈ 0.50 (Study 1 HiCo survey) | within ±.07 |
| `silence_voice_corr` | r = −.55 (Study 4) | within [−.65, −.45] |
| most-frequent IVT rule | "need solid data" (Study 2) | `need_data` is the top rule |

The silence–voice correlation is the run-level **time-averaged** per-agent value (written to `llm_meta.json` as `silence_voice_corr`): the per-step value in `metrics.csv` is the within-step VOICE/SILENCE snapshot (a hard dichotomy → −1), whereas the construct the paper reports is the graded overlap between distinct silence and voice tendencies.

## 2. CFA-style fit indices (`cfa_fit_indices.csv`)

The tool builds the N×5 binary IVT rule-activation matrix from `agents.csv` (the `active_rules` column), forms its correlation matrix, and fits a **single common factor** to it, reporting χ² / df / CFI / RMSEA / SRMR-like. Because the five rules fire *discriminantly* (low off-diagonal co-occurrence), a single factor fits poorly — operationalising the paper's Study-3 finding that the rules form a multi-rule (5-factor) structure (RMSEA = .05, CFI = .97 for the *correct* model) rather than collapsing into one general silence factor.

`factor-analyzer` is used for the loadings when available; a PCA fallback keeps the path running end-to-end under any compatible NumPy / scikit-learn versions.

The maximum off-diagonal rule co-occurrence (from `metrics.csv`) is reported alongside as the direct discriminant statistic; it stays below 0.50 (the design's discriminant-validity criterion).
