## 1. HiCo 50% calibration (rule mode)

Reproduce the paper's headline finding that roughly half of employees who hold an upward concern stay silent (Study 1's HiCo company-wide survey). Run the `rule` mode at the calibrated point (`ι̅ = 0.55`, `β_ι = 2.0`) and inspect `upward_silence_rate`:

```bash
cargo run --release -- run --llm-mode rule --t-max 60 --runs 30 --seed 42
cargo run --release -- reproduce --llm-mode rule --t-max 60 --runs 30
```

The steady-state `upward_silence_rate` lands near 0.50 (±.07).

## 2. IVT necessity ablation

Isolate the causal contribution of the Implicit Voice Theory term by contrasting `rule` (full `−β_ι ι` term) against `rule_no_ivt` (`β_ι = 0`):

```bash
cargo run --release -- ablation --modes rule,rule_no_ivt --seed-start 0 --seed-end 30
```

The IVT term raises mean upward silence by roughly the paper's incremental-validity ΔR² ≈ 0.12 (Study 4), with a large Cohen's d. Removing it drops upward silence below ~0.40 — operationalising the IVT term's necessity.

## 3. β_ι × ψ̄ sensitivity sweep

Map how the IVT main-effect coefficient and the psychological-safety axis jointly drive upward silence:

```bash
cargo run --release -- sweep \
    --beta-ivt-min 0.0 --beta-ivt-max 1.6 --beta-ivt-step 0.2 \
    --psafety-mean-values 0.3,0.5,0.7 --runs 30
uv run detert-tools visualize-sweep
```

`upward_silence_rate` rises monotonically with `β_ι`; raising ψ̄ tempers but does not eliminate the IVT-driven silence — the robustness of the IVT main effect under context (Study 4 Model 3).

## 4. LLM 5-rule self-reflection

Run the LLM mode (Ollama-first → OpenAI fallback) where the five IVT rules are embedded as a self-reflection layer in the decision prompt:

```bash
export OLLAMA_HOST=http://localhost:11434 OLLAMA_MODEL=llama3.1
cargo run --release -- run --llm-mode llm --cache-path runs/detert_cache.json --t-max 60
```

The LLM returns `{decision, motive, active_rules}`; the `rule_activation.csv` then records which rules the model reports firing. Re-running with the same arguments replays the warm cache (cache-hit → 100%).
