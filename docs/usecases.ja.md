## 1. HiCo 50% 較正（rule モード）

上位への懸念を抱える従業員の約半数が沈黙するという論文の中心的知見（Study 1 の HiCo 全社調査）を再現する．較正点（`ι̅ = 0.55`，`β_ι = 2.0`）で `rule` モードを実行し，`upward_silence_rate` を確認する：

```bash
cargo run --release -- run --llm-mode rule --t-max 60 --runs 30 --seed 42
cargo run --release -- reproduce --llm-mode rule --t-max 60 --runs 30
```

定常 `upward_silence_rate` は 0.50（±.07）付近に着地する．

## 2. IVT 必要性 ablation

`rule`（完全な `−β_ι ι` 項）と `rule_no_ivt`（`β_ι = 0`）を対比し，暗黙の発言理論項の因果寄与を分離する：

```bash
cargo run --release -- ablation --modes rule,rule_no_ivt --seed-start 0 --seed-end 30
```

IVT 項は平均 upward silence を論文の増分妥当性 ΔR² ≈ 0.12（Study 4）程度引き上げ，大きな Cohen's d を示す．除去すると upward silence は ~0.40 を下回り，IVT 項の必要性を操作化する．

## 3. β_ι × ψ̄ 感度スイープ

IVT 主効果係数と心理的安全性軸が upward silence をどう共同で駆動するかを図示する：

```bash
cargo run --release -- sweep \
    --beta-ivt-min 0.0 --beta-ivt-max 1.6 --beta-ivt-step 0.2 \
    --psafety-mean-values 0.3,0.5,0.7 --runs 30
uv run detert-tools visualize-sweep
```

`upward_silence_rate` は `β_ι` とともに単調増加する．ψ̄ を上げても IVT 駆動の沈黙は緩和されるが消えない — 文脈下での IVT 主効果の頑健性（Study 4 Model 3）．

## 4. LLM 5 ルール内省

5 つの IVT ルールを意思決定プロンプトの内省レイヤとして埋め込んだ LLM モード（Ollama 第一 → OpenAI フォールバック）を実行する：

```bash
export OLLAMA_HOST=http://localhost:11434 OLLAMA_MODEL=llama3.1
cargo run --release -- run --llm-mode llm --cache-path runs/detert_cache.json --t-max 60
```

LLM は `{decision, motive, active_rules}` を返し，`rule_activation.csv` がモデルの報告する発火ルールを記録する．同一引数で再実行すると warm cache を再生する（cache-hit → 100%）．
